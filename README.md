# 📊 CryptoDash

A modern cryptocurrency dashboard that provides real-time market data, interactive charts, favorites tracking, and currency conversion using public APIs.

---

## 🚀 Live Demo

👉 https://sindhukumarray.github.io/Crypto-dashboard/

---

## ✨ Features

* 🔍 Search cryptocurrencies (Bitcoin, Ethereum, etc.)
* 📊 Real-time price data using CoinGecko API
* 🚀 Top Gainers & 📉 Top Losers (24h)
* ⭐ Favorites system (click to add/remove)
* 📈 Interactive charts (Chart.js)
* 💱 Currency Converter (multi-currency support)
* 🌙 Dark / ☀️ Light mode toggle
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Grid)
* JavaScript (ES6 Modules)
* Chart.js
* CoinGecko API
* ExchangeRate API
* LocalStorage
