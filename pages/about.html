About CryptoDash

CryptoDash is a modern web-based dashboard that provides real-time cryptocurrency market data. It is designed to help users track prices, analyze trends, and explore the crypto market with ease.

📊 Features

  • Real-time cryptocurrency prices
  • Top gainers & losers tracking
  • Interactive price charts
  • Favorites system using local storage
  • Currency converter

⚙️ Technologies Used

  • HTML, CSS, JavaScript
  • Chart.js for visualization
  • CoinGecko API
  • LocalStorage

🎯 Purpose

The goal of CryptoDash is to provide a simple, fast, and user-friendly interface for monitoring cryptocurrency trends without requiring users to subscribe to premium tools.
